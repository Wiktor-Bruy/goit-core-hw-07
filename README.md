# goit-core-hw-07

This console application is a small assistant. It helps you manage your contact list. I worked on it while learning Python.

## What the app can do

- _create contacts and add them to your book_.
- _add phone numbers to contacts_.
- _change phone numbers for a specified contact_.
- _search for a phone number by contact name_.
- _add birthday date to contact_.
- _show the birthday date of the specified contact_.
- _display a list of all contacts_.
- _display a list of contacts to congratulate next week_.
